# Greg Klassen

A responsive single-page website built with modern web development practices.

## Features

- **Responsive Design** - Adapts to all screen sizes
- **Modern CSS** - SCSS with BEM methodology for maintainable code
- **Interactive UI** - Hover effects with state changes
- **Performance Focus** - Optimized assets and efficient styling
- **Accessible** - Semantic HTML structure
